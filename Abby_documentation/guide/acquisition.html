Traces acquisition¶

Acquisition setup¶

The acquisition setup depends on what you want to model. Here we are going to give instructions to acquire power traces from some STM32F0/STM32F1 boards but you should adapt to your hardware.

Target preparation¶

For power traces side channel using a power probe, some capacitors may be removed from the target to reduce filtering and get better quality traces. This operation can destroy the board or make it unstable. Be careful and do this at your own risk.

To find which capacitors to remove, you might refer to the board schematics and look at capacitors between ground and \(V_{DD}\). These capacitors are often located near the core.

STM32F030 and STM32F051 prepared by removing C18, C19, C20 and C21 capacitors.¶

STM32VL prepared by removing C18, C19, C20 and C21 capacitors.¶

BBC Micro:bit prepared by removing C18, C19 and C32 capacitors.¶

To output the external crystal oscillator clock on ST STM32F030 or STM32F051 Discovery boards, you need to bridge SB17. Then the clock will be outputted on PF0.

Power probe or resistor setup¶

Acquisition setup using a power probe.¶

You can either use a resistor or a current probe in series with the target. We used a stabilized external DC power supply. Remember to check the bandwidth of the different elements of your setup.

Oscilloscope setup¶

You should set the gain to capture the full dynamic range of the side-channel trace. We choose to input the target clock on the second channel to be able to isolate clock cycles.

Programmer setup¶

The STM32VL Discovery board has a ST-Link v1 that does not work well with OpenOCD. We can connect an external ST-Link v2 by disconnecting the 2 jumpers between the on-board ST-Link and the target, then following this:

Sample acquisition script¶

Abby includes modules to automate the flashing process of firmware and control the oscilloscope.

The following script acquires traces while executing specific code on target.

  1#!/usr/bin/env python3
  2
  3"""
  4Sample script to acquire traces.
  5"""
  6
  7import argparse
  8import os
  9import pathlib
 10import secrets
 11
 12import numpy as np
 13import pandas as pd
 14from serial import Serial
 15from tqdm import tqdm
 16
 17import abby
 18
 19
 20def main(opt):
 21    # Create destination folder if missing
 22    dest = pathlib.Path(opt.output).absolute()
 23    os.makedirs(dest, exist_ok=True)
 24
 25    with abby.oscilloscope.PS3000a() as ps:
 26        for algo in tqdm(opt.algorithm):
 27            # Build firmware for target then upload firmware to target board
 28            abby.firmware.pio_run(
 29                opt.board,
 30                algo,
 31                upload=True,
 32                debug=opt.debug,
 33            )
 34
 35            # Open serial port after flashing
 36            with Serial("/dev/ttyUSB0", baudrate=115200, timeout=1) as ser:
 37
 38                # Repeat acquisition with different input texts
 39                for _ in tqdm(range(opt.num)):
 40                    # Input text contains all input data for selected algorithm
 41                    if opt.input is not None:
 42                        # Read input text from file
 43                        input_text = bytes.fromhex(opt.input.readline())
 44                        input_size = algo.get_input_length() + 1
 45                        if len(input_text) != input_size:
 46                            raise IndexError(
 47                                f"\nInput text {input_text.hex()} does not "
 48                                f"match size {input_size}"
 49                            )
 50                    else:
 51                        # Random input text
 52                        input_text = b"\xAE"  # start byte
 53                        input_text += secrets.token_bytes(algo.get_input_length())
 54
 55                    # If file already exist, skip
 56                    output = dest / f"{opt.board}_{algo}_{input_text.hex()}.npy"
 57                    if output.is_file():
 58                        continue
 59
 60                    # Regenerate random code if necessary
 61                    if algo.name == "generated-code":
 62                        algo.seed = input_text
 63                        abby.firmware.pio_run(
 64                            opt.board,
 65                            algo,
 66                            upload=True,
 67                            debug=opt.debug,
 68                        )
 69
 70                    # Acquire trace
 71                    output_len = 1 + algo.msg_length  # +1 for header
 72                    _, trace, clock = ps.run_and_acquire(
 73                        input_text, output_len, ser, average=opt.average
 74                    )
 75
 76                    if not opt.no_downsample:
 77                        # Get the max of each cycle
 78                        cycles_indexes = abby.processing.find_cycles(clock)
 79                        downsampled_trace = np.zeros(len(cycles_indexes) - 1)
 80                        for i, c_index in enumerate(cycles_indexes[:-1]):
 81                            c_end = cycles_indexes[i + 1]
 82                            downsampled_trace[i] = np.max(trace[c_index:c_end])
 83                        downsampled_trace -= np.mean(downsampled_trace)
 84                        trace = downsampled_trace
 85
 86                    if not opt.no_crop:
 87                        # Crop NOP cycles from power trace
 88                        trace = abby.processing.crop_cycles(trace, threshold=0.005)
 89
 90                    # Save trace
 91                    np.save(output, trace)
 92
 93
 94if __name__ == "__main__":
 95    # Arguments parser
 96    parser = argparse.ArgumentParser(
 97        description="Acquire traces.",
 98    )
 99    parser.add_argument(
100        "-q",
101        "--quiet",
102        action="store_true",
103        default=False,
104        help="silent mode: hide info and warnings, overrides debug",
105    )
106    parser.add_argument(
107        "--debug",
108        action="store_true",
109        default=False,
110        help="debug mode: show debug messages",
111    )
112    parser.add_argument(
113        "-n",
114        "--num",
115        type=int,
116        default=1,
117        help="amount of traces for each algorithm, default to 1",
118    )
119    parser.add_argument(
120        "--average",
121        type=int,
122        default=50,
123        help="amount of traces averaged with same input data, default to 50",
124    )
125    parser.add_argument(
126        "-b",
127        "--board",
128        required=True,
129        choices=abby.firmware.environments,
130        help="target device to flash",
131    )
132    parser.add_argument(
133        "-a",
134        "--algorithm",
135        nargs="+",
136        default=[abby.firmware.blockcipher.GeneratedCode()],
137        choices=abby.firmware.blockcipher.blockciphers,
138        type=abby.firmware.get_blockcipher,
139        help="algorithm to flash, default to generated code",
140    )
141    parser.add_argument(
142        "--no_downsample",
143        action="store_true",
144        default=False,
145        help="disable downsampling",
146    )
147    parser.add_argument(
148        "--no_crop",
149        action="store_true",
150        default=False,
151        help="disable cropping of NOP instructions",
152    )
153    parser.add_argument(
154        "-i",
155        "--input",
156        nargs="?",
157        type=argparse.FileType("r"),
158        help="text file containing input to send, default to random",
159    )
160    parser.add_argument(
161        "-o",
162        "--output",
163        required=True,
164        help="destination folder for saved traces",
165    )
166    options = parser.parse_args()
167    abby.logger.setup_logger("abby", options.quiet, options.debug)
168    main(options)

Converting acquired traces to Riscure Inspector format¶

Riscure provide trsfile Python module to read and write to Riscure Inspector format. If you don’t want to work with Python and prefer Inspector, you can adapt the following piece of code to convert from Numpy to Inspector:

from trsfile import trs_open, Trace, SampleCoding, TracePadding, Header

MY_TRACES = []  # array of tuples (input_text as bytes, trace as np.array)
with trs_open("output.trs", "w",
    headers={Header.LABEL_X: 'Samples', Header.LABEL_Y: 'Power'},
    padding_mode=TracePadding.AUTO,
) as trs:
    for input_text, trace in MY_TRACES:
        trs.append(Trace(
            SampleCoding.FLOAT,
            trace,
            data=input_text,
        ))
