Emulators¶

class abby.emulator.ThumbulatorEmulator(fw_path)[source]¶

Bases: abby.emulator.base.Emulator

Thumbulator emulator for Cortex-M0.

This version of Thumbulator was patched by ELMO authors to add ELMO power model inside. It was also patched by Abby authors to output needed features.

This class depends on elmotrace Python module which wrap our modified version of ELMO as a Python module.

When using this emulator, some instruction related to GPIO might create unpredictable outputs. To circumvent this, we need a firmware compiled with ELMO flag.
