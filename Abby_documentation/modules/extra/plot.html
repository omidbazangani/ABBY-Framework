abby.plot.plot_fourier(trace, freq_estimated=8000000.0, freq_precision=1000.0, threshold=0.0002, sample_rate=500000000.0)[source]¶

Plot trace and Fourier transform on current Matplotlib figure.

The Fourier transform of trace should contain a spike at target clock frequency. You can use this function to inspect your target clock frequency.

For example:

>>> plt.figure(figsize=(16,10))
>>> abby.plot.plot_fourier(trace)
>>> plt.show()
Parameters

  • trace ([float] or np.ndarray) – side-channel trace to inspect

  • freq_estimated (float, optional) – estimated clock frequency in Hz, defaults to 8 MHz

  • freq_precision (float, optional) – clock frequency precision in Hz, defaults to 1 kHz

  • threshold (float, optional) – threshold line for spikes detection, defaults to 0.0002

  • sample_rate (float, optional) – trace sampling rate in Hz, defaults to 500 MHz

abby.plot.plot_ttest(ttest, x_axis=None, xlim=None, absolute=False, **kwargs)[source]¶

Plot t-test result on current Matplotlib figure.

This helper function adds the 4.5 threshold line (99.999% confidence) and do some common formatting.

Parameters

  • ttest ([float] or np.ndarray) – t-test result

  • x_axis ([float] or np.ndarray, optional) – X-axis used for plotting, defaults to simple range

  • xlim ((float, float), optional) – X-axis range to plot, defaults to full range

  • absolute (bool, optional) – plot absolute value of t-test, defaults to False
