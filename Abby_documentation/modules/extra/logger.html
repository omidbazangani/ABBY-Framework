Logger¶

Pretty logging for nice command line programs.

When using tqdm (progress bars) with Python logging, text output is broken. This module implements a custom logger that solves this issue.

This logger also colors output and set the right verbosity level.

abby.logger.setup_logger(name='abby', quiet=False, debug=False) → logging.Logger[source]¶

Create Python logging.Logger with colors and tqdm support.

If you want to apply this logger configuration to abby submodules, you should name it abby as abby.something will inherit configuration.

Default verbosity is INFO, WARNING, ERRORS and CRITICAL. If quiet, show only ERRORS and CRITICAL. If debug, show everything.

Parameters

  • name (str, optional) – name of the logger, should be the root of all other logger

  • quiet (bool, optional) – only print warning and errors, defaults to False

  • debug (bool, optional) – print debugging information, defaults to False

Returns

custom logger

Return type

logging.Logger
